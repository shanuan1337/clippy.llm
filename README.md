# 📎 Клиппи — офисный помощник для Windows

Классический помощник из Microsoft Office 97, который следит за активным окном и комментирует происходящее с помощью локальной мультимодальной LLM (Qwen 3.6 35B A3B).

## Возможности

- Анимированная скрепка поверх всех окон (GIF)
- Речевое облачко с хвостиком
- Прямая отправка скриншотов в LLM (без отдельного OCR)
- Генерация саркастичных комментариев через LLM
- Автоматическое сохранение скриншотов с ответами Клиппи
- Перетаскивание окна мышью

## Структура проекта

```text
C:\Programs\clippy.llm
├── clippy_overlay.py  # Главный скрипт
├── clippy.gif         # Анимация скрепки (GIF)
└── README.md          # Этот файл
```

> **Важно:** Пути в коде абсолютные (`C:\Programs\clippy.llm\`). Весь проект должен лежать именно в этой папке.

## Установка зависимостей

Откройте командную строку от имени администратора и выполните:

```cmd
cd C:\Programs\clippy.llm
python -m pip install --upgrade pip
python -m pip install PyQt5 mss pywin32 requests pillow
```

## Настройка параметров

Откройте `clippy_overlay.py` и найдите в начале файла секцию **НАСТРОЙКИ**:

```python
# ========== НАСТРОЙКИ ==========
LLM_API_URL = "http://192.168.0.18:8000/v1/chat/completions"
LLM_TOKEN = "token-abc123"
LLM_MODEL = "Qwen3.6-35B-A3B-Q5-256K"
INTERVAL_SECONDS = 15
SCREENSHOT_DIR = "C:/Programs/clippy.llm/Screenshots"
```

### Регулировка частоты появления фраз

Измените `INTERVAL_SECONDS`:

```python
INTERVAL_SECONDS = 30   # Каждые 30 секунд (меньше — чаще, больше — реже)
```

### Регулировка времени показа облачка

В функции `show_bubble` измените таймаут (в миллисекундах):

```python
QTimer.singleShot(15000, self.bubble.hide)   # 15000 = 15 секунд
```

## Запуск

cd C:\Programs\clippy.llm
